About

Who I am

I'm a Education(ICT) student at Mzuzu University, Malawi. I've always been curious about how software works under the hood — and more importantly, how it can solve real problems for real people.

I started with Java and Python through my coursework, then taught myself HTML, CSS, and JavaScript because I wanted to see my ideas come alive in the browser. Now I'm leveling up with React and React Native to build proper cross-platform apps.

I'm the kind of developer who breaks things first and figures out why later. I believe building is the best way to learn — every bug is a lesson, every project a stepping stone.

When I'm not coding, I'm probably explaining something to a classmate, reading about new tech, or brainstorming the next thing to build.
